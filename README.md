# Competitive Programming and Contests 25-26
## Introduction
This repository contains all the hands-on assignments and the mandatory exercises (the ones in bold on the following page: https://pages.di.unipi.it/rossano/competitive/)
for the Competitive Programming and Contests course at the University of Pisa, academic year 2025–2026.
